# get_id_bot

## Usage
```
npm start
```
## Installation
* Clone the repository
* Download the required packages with ```npm install```
* Set your Telegram bot token inside a .env file

## Docker setup
* ```cd get_id_bot```
* ```docker build -t eliapolloniato/get_id_bot .```
* ```docker run -d eliapolloniato/get_id_bot```
